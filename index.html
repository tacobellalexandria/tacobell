unseen life
using poseNet to see the unseen
link here

imagine if a code was created to break the boundaries of the 3D and allow us to see what the naked eye cannot. As mere humans, we and our capabilities of sight are limited to the third dimension. However this does NOT mean that what we can’t see isn’t there. “Unseen Life” uses an AI machine learning tool that tracks body movement and body parts to help us see the life our eyes cannot- through the screen. Along with any basic movement, users are able to trigger the detection of “life” and see how other interdimensional beings exist on the human body.

design and composition

  • i decided to use poseNet because it worked well with my narrative
  • users use the camera to look at themselves and move around to trigger life detection
  • this program wroks in real time, so no matter what the user does, each visual will be unique and different

technical components

  • the code incorporates the use of for loops and if statesments along with scale which is what we learned in class and applies it to the poseNet tool
